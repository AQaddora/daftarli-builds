# Installation Guide

## Choose Your Platform

### Android

#### System Requirements
- Android 7.0 (Nougat) or higher
- 100 MB free storage
- Optional: Internet for first-time activation

#### Installation Steps

1. **Download APK**
   - Visit [Downloads Page](../README.md)
   - Click "Download for Android (APK)"
   - Wait for download to complete

2. **Enable Installation from Unknown Sources**
   - Go to Settings > Security
   - Enable "Unknown Sources" or "Install Unknown Apps"
   - Grant permission to your browser/file manager

3. **Install APK**
   - Open Downloads folder
   - Tap the Daftarli APK file
   - Tap "Install"
   - Wait for installation to complete

4. **Open Daftarli**
   - Find Daftarli icon on home screen or app drawer
   - Tap to open
   - Complete first-time setup

#### Troubleshooting Android

**Problem**: Can't install APK
- **Solution**: Make sure "Unknown Sources" is enabled in Settings

**Problem**: "App not installed" error
- **Solution**: Uninstall any previous version first, then retry

**Problem**: App crashes on startup
- **Solution**: Clear cache, restart device, reinstall

### iOS (Progressive Web App)

#### System Requirements
- iOS 12.2 or higher
- Safari browser
- Internet connection

#### Installation Steps

1. **Open Safari**
   - Launch Safari browser on your iPhone/iPad
   - Go to: https://daftarli.com

2. **Install as App**
   - Tap the Share button (square with arrow)
   - Scroll down and tap "Add to Home Screen"
   - Edit name if desired (default: "Daftarli")
   - Tap "Add"

3. **Open Daftarli**
   - Find Daftarli icon on home screen
   - Tap to open
   - Works like a native app!

#### Troubleshooting iOS

**Problem**: Don't see "Add to Home Screen"
- **Solution**: Make sure you're using Safari, not Chrome or another browser

**Problem**: App doesn't work offline
- **Solution**: Open the app once while online to cache data

### Windows Desktop

#### System Requirements
- Windows 10 or higher (64-bit)
- 200 MB free storage
- Optional: Internet for updates

#### Installation Steps

1. **Download Installer**
   - Visit [Downloads Page](../README.md)
   - Click "Download for Windows"
   - Save the .exe file

2. **Run Installer**
   - Double-click the downloaded .exe file
   - If Windows SmartScreen appears:
     - Click "More info"
     - Click "Run anyway"
   - Click "Next" to start installation

3. **Choose Install Location**
   - Default: C:\Program Files\Daftarli
   - Click "Next"

4. **Create Desktop Shortcut**
   - Check "Create desktop shortcut" (recommended)
   - Click "Install"
   - Wait for installation

5. **Launch Daftarli**
   - Click "Finish" to launch immediately
   - Or find in Start Menu > Daftarli

#### Troubleshooting Windows

**Problem**: Windows blocks installation
- **Solution**: Click "More info" then "Run anyway"

**Problem**: Installer won't run
- **Solution**: Right-click installer, select "Run as administrator"

**Problem**: App won't start
- **Solution**: Check Windows updates, install Visual C++ Redistributable

### macOS Desktop

#### System Requirements
- macOS 10.13 (High Sierra) or higher
- 200 MB free storage
- Optional: Internet for updates

#### Installation Steps

1. **Download DMG**
   - Visit [Downloads Page](../README.md)
   - Click "Download for macOS"
   - Save the .dmg file

2. **Open DMG**
   - Double-click the downloaded .dmg file
   - A window will open showing Daftarli

3. **Install Application**
   - Drag Daftarli icon to Applications folder
   - Wait for copy to complete
   - Eject the DMG (right-click > Eject)

4. **First Launch**
   - Open Applications folder
   - Double-click Daftarli
   - If macOS blocks it:
     - Right-click Daftarli
     - Select "Open"
     - Click "Open" in the dialog
   - Or: System Preferences > Security & Privacy > "Open Anyway"

#### Troubleshooting macOS

**Problem**: "Can't be opened because it's from an unidentified developer"
- **Solution**: Right-click app, select Open, then click Open

**Problem**: App is damaged
- **Solution**: In Terminal, run: `xattr -cr /Applications/Daftarli.app`

**Problem**: Permission denied
- **Solution**: Check disk permissions, try copying again

## First-Time Setup

After installation, follow these steps:

### 1. Create Account

- Click "Start Free" or "Register"
- Enter your details:
  - Business name
  - Your name
  - Email
  - Phone
  - Password
- Accept terms of service
- Click "Create Account"

### 2. Business Setup

- Enter business information:
  - Full business name
  - Business type (retail, restaurant, etc.)
  - Address
  - Tax ID (if applicable)
- Upload logo (optional)
- Set currency
- Choose language

### 3. Initial Configuration

- Review default account structure
- Add your first products (if retail)
- Set up customer accounts (if needed)
- Configure receipt settings
- Enable offline mode

### 4. Start Using

- Explore the dashboard
- Make your first sale
- Record an expense
- Check reports

## Uninstallation

### Android
1. Settings > Apps > Daftarli
2. Tap "Uninstall"
3. Confirm

### Windows
1. Settings > Apps > Apps & features
2. Find Daftarli
3. Click "Uninstall"

### macOS
1. Open Applications folder
2. Drag Daftarli to Trash
3. Empty Trash

### iOS PWA
1. Long-press Daftarli icon
2. Tap "Remove App"
3. Confirm

## Data Backup

**Before uninstalling**, backup your data:

1. Open Daftarli
2. Go to Settings > Backup & Restore
3. Click "Export All Data"
4. Save the backup file safely
5. You can restore later by clicking "Import Data"
