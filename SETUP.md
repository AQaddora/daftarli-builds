# Releases Repository Setup

This directory is a separate Git repository for hosting Daftarli release binaries.

## Initial Setup

Run this once to initialize and connect to the remote repository:

```bash
cd releases
git init
git lfs install
git remote add origin https://github.com/AQaddora/daftarli-builds.git
git add .
git commit -m "Initial commit: Setup releases repository"
git branch -M main
git push -u origin main
```

## Repository URL

**Public Repository**: https://github.com/AQaddora/daftarli-builds

**Raw Download Base URL**: https://github.com/AQaddora/daftarli-builds/raw/main/

## Git LFS

This repository uses Git LFS (Large File Storage) for binary files. Make sure you have Git LFS installed:

```bash
# macOS
brew install git-lfs

# Ubuntu/Debian
sudo apt-get install git-lfs

# Windows
# Download from: https://git-lfs.github.com/
```

## File Naming Convention

All release files follow this pattern:
```
daftarli-{version}-{platform}.{ext}
```

Examples:
- `daftarli-1.0.0-android-apk.apk`
- `daftarli-1.0.0-electron-mac.dmg`
- `daftarli-1.0.0-electron-win.exe`

## Deployment

Use the unified deployer script from the main repository:

```bash
# From the main repository root
node scripts/deploy-all-standalones.mjs

# Build specific platform
node scripts/deploy-all-standalones.mjs --platform android-apk

# Build without deploying
node scripts/deploy-all-standalones.mjs --build-only

# Deploy without building (use existing binaries)
node scripts/deploy-all-standalones.mjs --skip-build
```

## Manual Upload

If you need to manually add a release:

1. Copy the binary to this directory
2. Rename it following the naming convention
3. Update `latest.json` manifest
4. Commit and push:

```bash
git add .
git commit -m "Release: v1.0.0"
git push origin main
```

## latest.json Manifest

The `latest.json` file contains metadata about the latest release:

```json
{
  "version": "1.0.0",
  "releaseDate": "2026-01-22T00:00:00.000Z",
  "builds": {
    "android-apk": {
      "fileName": "daftarli-1.0.0-android-apk.apk",
      "size": 50000000
    },
    "electron-mac": {
      "fileName": "daftarli-1.0.0-electron-mac.dmg",
      "size": 100000000
    }
  },
  "downloadBaseUrl": "https://github.com/AQaddora/daftarli-builds/raw/main"
}
```

This file is automatically generated by the deployment script.
